# Homebrew Cask formulae for Makaio

## Install (stable)

```bash
brew tap makaio-ai/makaio
brew install --cask makaio
```

## Install (canary)

```bash
brew tap makaio-ai/makaio
brew install --cask makaio-canary
```

Canary tracks the latest pre-release builds. It installs the same `Makaio.app` and
`makaio` CLI — use it to test upcoming features before they hit stable.

## What you get

- **Makaio.app** in `/Applications`
- **`makaio` CLI** in `/usr/local/bin`

## Upgrade

```bash
brew upgrade --cask makaio
# or
brew upgrade --cask makaio-canary
```

## Uninstall

```bash
brew uninstall --cask makaio
# or
brew uninstall --cask makaio-canary
```
